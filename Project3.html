Processing is a coding Language based on Java designed to allow coder to visually express themselves. It allows for many different interaction, drawing and animations.

In 2019 we had a Bootcamp program where we put through intensive lesson.

The Goal:

Combine everything we had learned so far in the previous assignments. We had free choice of what we had to do but we need to follow set criteria which are listed below.

    Make sure you use the following elements :

  • Shapes and colours
  • User input
  • Conditionals
  • Functions
  • Loops
  • Arrays

The Idea

I wanted to showcase how voice can be used as an interactive tool. I wanted to be able to demonstrate that technology has gone that far that with the power of the voice that there is a lot more you can do than just talking to Google.

My inspiration struck me when I thought of my hometown how everything so dark and you can see every single star. I also knew I wanted to create something that would challenge me. I saw a 3D sphere and it struck me how I could create a solar system and by using the voice you can line all of them up. The way this works is by setting the x & y position of each 3D sphere at lower one then the point of origin. By input sound from the microphone, you will be adding x value to the x & y positions ending up with a sum the same as the point of null/origin.
